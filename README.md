## About QuickMeds

QuickMeds is an online pharmacy platform managing orders, inventory,
and patient data across India. This mini project is built by a team
of 8 data engineering students, simulating a real-world data pipeline
using AWS S3, Databricks, PySpark, and GitHub.

## Repository Structure
```

├── data/        # Sample CSV files

├── notebooks/   # Databricks notebooks
```

## Goals

- Version control for all pipeline scripts and notebooks
- Team collaboration via GitHub
- Pharmacy compliance through traceable change history
- Reusable Python utility modules across all pipeline blocks
