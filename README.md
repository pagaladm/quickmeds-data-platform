# quickmeds-data-platform
QuickMeds is an online pharmacy platform managing  orders, inventory, and patient data across India.  This mini project is built by a team of 8 data  engineering students, simulating a real-world  data pipeline using AWS S3, Databricks, PySpark,  and GitHub.

QuickMeds Data Platform
Author: Diksha Kumari Sah
Batch: AWS Batch

Company Description
QuickMeds is a modern pharmacy chain that leverages data engineering to streamline operations, ensure regulatory compliance, and improve patient outcomes. The QuickMeds Data Platform centralizes all pipeline scripts, notebooks, and utilities to enable collaborative, auditable, and compliant data workflows across the organization.

Repository Structure
quickmeds-data-platform/
├── data/               # Sample CSV files for testing and development
├── scripts/            # Python scripts for data pipelines and utilities
├── notebooks/          # Exported Databricks notebooks
├── docs/               # Architecture diagrams and documentation
├── README.md           # Project overview
└── .gitignore          # Ignored files and folders

Goals

Maintain all pipeline scripts and notebooks under version control
Enable team collaboration via GitHub
Meet pharmacy compliance requirements through traceable change history
Provide reusable Python utility modules across all pipeline blocks
